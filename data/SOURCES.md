# 데이터 출처 (Data Sources)

## 베이스 레시피 데이터셋 (breadth layer)
- **Hieu-Pham/kaggle_food_recipes** — https://hf.co/datasets/Hieu-Pham/kaggle_food_recipes
  - License: **CC-BY-SA-3.0** (출처표기 + 동일조건변경허락)
  - 약 13.5K 레시피 (Title/Ingredients/Instructions), 주로 서양식. 양식·이탈리안·프렌치 폭 보강용.
  - 원출처: Kaggle "Food Ingredients and Recipe Dataset with Images".

## 셰프 페르소나·메뉴 (chef-style layer)
- 8인 셰프의 요리 스타일·레스토랑·대표 메뉴는 **공개된 사실**(언론·레스토랑 정보)을 바탕으로 정리.
  - 최현석: 엘본 더 테이블 / 박은영: 누와(NUWA) / 샘킴: 오스테리아·뜨라또리아 샘킴 /
    정호영: 우동 카덴 / 손종원: 이타닉 가든·라망 시크레 / 윤남노: 디핀 / 권성준: 비아 톨레도 파스타 바.
- 정통 요리(까르보나라·마파두부·오야코동 등)는 널리 알려진 사실 기반.

> 추천 시 원문 레시피를 그대로 제공하지 않고, AI가 사용자 요구·셰프 페르소나에 맞게 변형해 제안한다.
